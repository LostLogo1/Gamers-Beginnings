Parts List

Home Parts Win a pc

Case

This is the main component that holds everything together which enables you to wire it up.

motherboard

Circuit board that houses all the main pc components like cpu and gpu.

CPU

This is the "brains" of the pc as this is what handles the inputs and outputs of the user.

CPU Cooler

Main cooling fuction for the cpu to keep it within operating ranges so nothing fails and stops pc from working.

RAM

This is short term memory that is needed in order to run apps or open documents. This is emptied when you shut off the pc.

Storage

Records all the data that wishes to be saved like documents and games.

GPU

This handles all the grahpical inputs for games/videos.

Power Supply

Supplys the pc with power that is supplied from the sockets in the wall from your home or building.
